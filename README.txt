Wedding website starter

Files:
- index.html -> all page content and links
- styles.css -> all styling
- assets/ -> images used by the page

Where to edit:
1. Names, date, RSVP line, venue address:
   index.html -> SECTION 1: Invitation / Landing

2. Venue image:
   replace assets/venue.jpg with another image using the same file name,
   or change the background image path in styles.css under .venue-section

3. Love story text:
   index.html -> SECTION 3: Love story

4. Timeline:
   index.html -> section id="timeline"

5. Ceremony / Reception / Dress Code:
   index.html -> section id="details"

6. Registry links:
   index.html -> section id="registry"
   Replace the button href values with the real links.

7. Contact details:
   index.html -> section id="contact"

Notes:
- The current couple images were cropped from the design screenshot you shared,
  so they can be replaced later with the real photos.
- The site is static HTML/CSS, so it is easy to upload to GitHub Pages, Netlify, or Vercel.
